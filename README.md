# UAS-PROJECT_PAWSHOP
PAWSHOP

PAWSHOP is a simple pet shop e-commerce website developed as a Final Semester Project (UAS). This project was built using HTML, CSS, and JavaScript to demonstrate the implementation of front-end web development concepts, including responsive layouts, interactive user interfaces, and basic shopping cart functionality.

**About the Project**

PAWSHOP is designed to provide a convenient online shopping experience for pet owners. The website offers various pet-related products such as pet food, toys, accessories, and grooming supplies through a clean and user-friendly interface.

The purpose of this project is to apply the fundamental concepts of web development while creating a functional and visually appealing e-commerce website.

**Features
**
- Home page with promotional banner
- Product catalog with categories
- Product detail display
- Shopping cart functionality
- User registration and login
- Checkout page
- Order history
- User profile page
- Responsive design for desktop and mobile devices

**Technologies Used
**
- HTML
- CSS
- JavaScript
- php
